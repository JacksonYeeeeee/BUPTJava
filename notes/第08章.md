# 第八章 对象和类
## 基础知识
# 对象的定义
* OOP = Object Oriented Programming （只是面向对象相关技术的一个阶段，还有OOA OOD）
* 理解对象
    * 对象是一个实体，使用class来定义
    * 对象的状态/特征/属性（state/property/attribute），也即对象的成员
    * 对象的行为（behavior/action），也即对象的方法
    * 构造函数是为了对象的自动构造而设计
* Java中，所有函数都是类的成员函数
